# 샘플 입력 — 라이브 데모용

이 한 줄을 SKILL 호출 시 입력하면 5개 산출물이 모두 나온다.

```
주제: AI 시대의 1인 사이드프로젝트 — 8시간 안에 동작하는 무엇이든
기간: 8시간 (당일 09:00~17:00)
참가자 수: 30명 (개인전)
도구 제약: 자유 (Claude / Codex / Cursor / Copilot 모두 OK, prompt 공개 권장)
심사 기준 가중치: README 20%, 코드 25%, AI 활용도 25%, 완성도 20%, 차별성 10%
```
