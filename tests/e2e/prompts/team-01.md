## 만들 프로덕트 — 단어 빈도 카운터 CLI

표준 입력 (stdin) 으로 받은 텍스트의 단어 빈도를 세서 상위 N개를 출력하는 CLI 도구.

요구 사항:
- 한국어 / 영어 모두 처리
- 정렬 옵션 (빈도순 / 알파벳순)
- 최소 빈도 필터 (`--min`)
- 상위 N개 (`--top N`, 기본 10)
- 출력 형식: `단어 빈도` 줄 단위

언어 자유 (Python / Bash / Go 등 익숙한 것).
